Usage: nmap [Scan Type(s)] [Options] [target specification]

HOST DISCOVERY:

nmap -sL [Target] : List Scan - simply list targets to scan
nmap -sn [Target] : Ping Scan - disable port scan
nmap -Pn [Target] : Treat all hosts as online -- skip host discovery

PORT SPECIFICATION AND SCAN ORDER:

nmap -p [port ranges] [Target] : Only scan specified ports
Ex: -p22; -p1-65535; -p U:53,111,137,T:21-25,80,139,8080,S:9

SERVICE/VERSION DETECTION:

nmap -sV [Target]

SCRIPT SCAN:

nmap -sC [Target]

OS DETECTION:

sudo nmap -O [Target]

OUTPUT:

nmap [Target] -oN outputfile

EXAMPLES:

nmap -v -A scanme.nmap.org
nmap -v -sn 192.168.0.0/16 10.0.0.0/8
nmap -v -iR 10000 -Pn -p 80
nmap -sC -vvv -sV [Target]
